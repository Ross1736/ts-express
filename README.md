Instala ts
npm i -D typescript

Instala tipos
npm install -D @types/express @types/dotenv

Inicia ts
npx tsc --init

Compilar
npx tsc

Ejecutar
node dist/index.js

Instalar eslint
npm install --save-dev eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin

Iniciar eslint
npx eslint --init
